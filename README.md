# Website RAG System

## Project Overview
This project implements a Retrieval-Augmented Generation (RAG) pipeline using website data scraped from Sri Eshwar College of Engineering.

## URLs Scraped
1. https://sece.ac.in/
2. https://sece.ac.in/department-computer-science-engineering-2/
3. https://sece.ac.in/placement-training/

## Workflow
- Web Scraping using BeautifulSoup
- CSV Storage
- Text Chunking
- Embedding Generation
- ChromaDB Vector Database
- Retrieval
- Groq LLM Generation

## Technologies Used
- Python
- BeautifulSoup
- Pandas
- Sentence Transformers
- ChromaDB
- Groq
